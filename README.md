# Homebrew Tap

Custom Homebrew formulae by Alexander Loth.

## Installation

```bash
brew tap aloth/tap
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [olcli](Formula/olcli.rb) | [Overleaf CLI](https://github.com/aloth/olcli) - sync LaTeX projects from the command line |
| [cred1](Formula/cred1.rb) | [CRED-1](https://github.com/aloth/cred-1) domain credibility checker - library and CLI |

## Usage

```bash
# Install olcli
brew install aloth/tap/olcli

# Install cred1
brew install aloth/tap/cred1

# Or after tapping:
brew tap aloth/tap
brew install olcli
brew install cred1
```
